# Michigan & UP Family Road Trip

An interactive itinerary, packing list, and travel tracker for a family road trip through Chicago, Wisconsin, and Michigan's Upper Peninsula.

Built with React, Vite, TypeScript, and Tailwind. Includes an interactive trip map, todo checklist, and quick notes powered by Gemini.

## Run locally

```bash
npm install
cp .env.example .env.local   # add your GEMINI_API_KEY
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run preview
```
